# TetrisGame
### Домашнее задание: Игра

### Цель:
В процессе выполнения ДЗ вы получите опыт создания игр.


#### Описание/Пошаговая инструкция выполнения домашнего задания:
Написать несложную игру, на выбор:</p>
тетрис,</p>
сапёр,</p>
2048,</p>
крестики-нолики,</p>
четыре-в-ряд,</p>
ping-pong,</p>
space invaders,</p>
pacman</p>

Требования:</p>

Создано игровое приложение с главным меню и основным gameplay loop.</p>
Бонусные баллы за графическое оформление с использованием ассетов.</p>
Бонусные баллы за звуковое оформление.</p>
Бонусные баллы за таблицу лидеров.</p>
Код компилируется без предупреждений с ключами компилятора -Wall -Wextra -Wpedantic -std=c11.</p>
